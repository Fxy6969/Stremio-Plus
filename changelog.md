# Changelog

## v1.1.5
Forked from [stremio-enhanced-community v1.1.5](https://github.com/REVENGE977/stremio-enhanced-community) by REVENGE977.

**Changes in this fork:**

- Custom popup menu system ported and integrated into the Electron app

- Data directory renamed to `stremio-plus`
- All upstream features retained — see the [upstream changelog](https://github.com/REVENGE977/stremio-enhanced-community) for full history through v1.1.5

For upstream feature history (v0.3 through v1.1.5), see the [original project](https://github.com/REVENGE977/stremio-enhanced-community).
